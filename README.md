# Hotel Health Clinic V3

A professional hotel technology and marketing audit tool designed to identify revenue leakage and provide AI-powered strategic assessments.

![Hotel Health Clinic](https://picsum.photos/seed/hotel-tech/1200/600)

## 🚀 Features

- **Digital Health Audit**: A comprehensive questionnaire covering Booking Engines, Channel Managers, CRM, and more.
- **Real-time AI Analysis**: Powered by Google Gemini 3 Flash, providing urgent, clinical assessments of technical gaps.
- **Direct Revenue Strategy**: Focused on reducing OTA dominance and increasing high-margin direct traffic.
- **PDF Export**: Generate and download professional reports for stakeholders.
- **Multi-language Support**: Available in English, Italian, and Spanish.

## 🛠️ Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS 4
- **AI**: Google Gemini API (`@google/genai`)
- **Charts**: Recharts
- **Icons**: Lucide React
- **Animations**: Framer Motion (via `motion/react`)

## 📋 Prerequisites

- Node.js (v20 or higher)
- npm or yarn
- A Google Gemini API Key (get one at [aistudio.google.com](https://aistudio.google.com/))

## ⚙️ Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/hotel-health-clinic.git
   cd hotel-health-clinic
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   Create a `.env` file in the root directory and add your Gemini API key:
   ```env
   VITE_API_KEY=your_gemini_api_key_here
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```

## 🏗️ Building for Production

To create a production build:
```bash
npm run build
```
The output will be in the `dist/` directory.
