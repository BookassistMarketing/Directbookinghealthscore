## Current Gemini Prompt Architecture

### Hotel Tech Audit — `generateStrategicAnalysis`
- Model: `gemini-3-flash-preview`
- Input: score %, list of GAP items (question text + category), optional site URL
- Output: ~200 words, 4 mandatory sections, fully translated
- Fallback: `generateLocalAnalysis` (local static text) after 6 retries on 429/503

### AI Visibility Audit — `generateAiReadinessReport`
- Model: `gemini-3-flash-preview`
- Input: hotel website URL (fetched via `urlContext` tool)
- Output: structured markdown report — header, 4 content sections, no AI-generated Bookassist pitch
- Anti-injection rules enforced at top of system prompt (highest priority)
- No fallback — throws on exhausted retries
